# 多端购物商城产品规划（v0.3.0）

## 本期目标
将“简陋演示”升级为可展示的高完成度商城前台，重点放在功能深度、页面完成度与视觉统一。

## v0.3.0 交付范围（已完成）
1. **首页重构**
   - 顶部导航 + 搜索 + 自动轮播 Banner
   - 活动专区、分类宫格、热门榜单、新人专区
   - 高级商品卡（评分/销量/标签/原价划线/折扣）
   - 响应式适配（移动/PC）

2. **商品链路补齐**
   - 商品详情页：图文、规格选择、库存、评价预览
   - 加入购物车反馈（toast）
   - 立即购买流程直达结算

3. **购物车与结算升级**
   - 购物车：数量增减、全选、删除、删除选中、失效态
   - 结算：地址管理入口、优惠券、运费、备注、价格明细
   - 下单成功页 + 订单入口

4. **个人中心**
   - 个人主页（头像、会员信息、订单入口）
   - 订单页（全部/待付款/待发货/已完成）
   - 收藏页、收货地址管理页

5. **视觉系统**
   - 设计 token：品牌色、阴影、圆角、基础字体体系
   - 统一组件：Button/Card/Tag/Empty/Skeleton
   - 关键反馈：loading、empty、error、toast

6. **Mock 与接口契约**
   - 新增 `/api/home`、`/api/products/{id}`、`/api/reviews`
   - `/api/orders` 支持状态过滤
   - 契约与页面保持稳定，便于真实后端替换

7. **质量保障**
   - 新增 5+ 测试（组件 + 交互流程 + store 逻辑）

## 下一步建议
- 接入真实后端鉴权与用户态
- 强化订单详情、物流、售后流程
- 增加埋点、A/B 与运营策略模块
